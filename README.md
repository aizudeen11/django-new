# 🌐 Django Web Application

This project is a **Django-based web backend application** that includes user authentication, database models, dynamic views, and a sentiment analysis feature.

It demonstrates full-stack backend development using Django, including handling user data, rendering templates, and integrating a simple machine learning feature.

---

## 🚀 Features

### 🔐 Authentication System
- User registration
- User login
- User logout
- Session-based authentication

### 🗄️ Database & Models
- Custom Django models for data storage
- ORM-based database interactions
- Scalable structure for future features

### 🎨 Templates & Views
- Multiple HTML templates
- Dynamic rendering using Django views (`views.py`)
- Organized routing with clean URL structure

### 😊 Sentiment Analysis Page
- User inputs text
- System analyzes sentiment
- Output displayed as:
  - **Positive**
  - **Negative**
